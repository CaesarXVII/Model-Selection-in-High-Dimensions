This is the student repository for the course "Model Selection for prediction, classification & model building in very large settings (Big Data)".

It is given in the spring semester 2018 at the University of Geneva for the Master in Statistics by prof. Maria-Pia Victoria-Feser with Guillaume Blanc and Cesare Miglioli as assistants.

In this repository all the projects and the contents created by the students will be stored for evaluation.
